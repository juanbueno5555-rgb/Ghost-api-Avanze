\begin{abstract}
This document presents the complete set of architecture diagrams for \textbf{Tech Cup}, a distributed platform built upon an API Gateway and eleven independent microservices. It covers the system context, component organisation, hexagonal architecture of the Auth service, authentication sequence, and CI/CD deployment pipeline.
\end{abstract}
